# 黎曼可积积分出题器

一个无依赖的小型 Web 应用，用来动态生成带显式初等函数解的积分题，并在后台通过本地 Claude Code CLI 自动完成命题与解题。

## 功能

- 支持 `定积分` 与 `不定积分` 两种模式
- 默认由模型一次性完成“出题 + 解题”，避免题目和答案不匹配
- 通过结构化 JSON 输出约束题目质量，避免特殊函数、数值近似和不可显式积分的情况
- 每次命题会随机挑选题型模板，并避开与上一题过于相似的题
- 当模型输出不合法、超时或与上一题太像时，服务端会自动重试
- 默认通过本地 `Claude Code` CLI 非交互求解
- 页面默认隐藏答案，点击“展示答案”后统一展开：
  - 模型生成的传统方法
  - 模型生成的技巧方法
  - 发给模型的动态命题提示词
  - 模型返回内容与原始结构化输出

## 启动

```bash
npm run dev
```

然后打开 [http://localhost:4173](http://localhost:4173)。

## 测试

```bash
npm test
```

## 环境要求

- 需要本机已安装并登录 `Claude Code`
- 默认直接调用命令 `claude -p`
- 如果另一台机器的命令名或路径不同，可设置环境变量 `CLAUDE_BIN`

示例：

```bash
CLAUDE_BIN=claude npm run dev
```

或：

```bash
CLAUDE_BIN=/opt/homebrew/bin/claude npm run dev
```
